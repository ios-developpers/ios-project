Git Config
==========

Config git on your computer

	git config --global user.name name
	git config --global user.email email@email.fai


Git Push & Pull
===============

Commit your changes

	git add file1 file2
	git commit -m"your message here"
	git push

Pull other programmers changes

	git pull
	git fetch
